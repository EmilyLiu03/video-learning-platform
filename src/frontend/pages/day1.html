🏗️ HTML - 网页的骨架

HTML（HyperText Markup Language）是构建网页的基础语言，就像建筑的框架一样，为网页提供结构和内容。

基本结构

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
<body>
    <h1>欢迎来到我的网页</h1>
    <p>这是一个段落。</p>
</body>
</html>

常用标签

  • <h1>-<h6> - 标题标签
  • <p> - 段落标签
  • <div> - 容器标签
  • <span> - 行内容器
  • <a> - 链接标签
  • <img> - 图片标签

🎨 CSS - 网页的美妆师

CSS（Cascading Style Sheets）负责网页的样式和布局，让网页变得美观和吸引人。

基本语法

/* 选择器 { 属性: 值; } */
h1 {
    color: #333;
    font-size: 24px;
    text-align: center;
}

.container {
    width: 80%;
    margin: 0 auto;
    padding: 20px;
}

常用属性

  • color - 文字颜色
  • background-color - 背景颜色
  • font-size - 字体大小
  • margin - 外边距
  • padding - 内边距
  • border - 边框

⚡ JavaScript - 网页的魔法师

JavaScript为网页添加交互性和动态功能，让静态的网页变得生动有趣。

基本语法

// 变量声明
let message = "Hello, World!";
const pi = 3.14159;

// 函数定义
function greet(name) {
    return "你好, " + name + "!";
}

// 事件处理
document.getElementById('button').addEventListener('click', function() {
    alert('按钮被点击了!');
});

核心概念

  • 变量 - 存储数据的容器
  • 函数 - 可重复使用的代码块
  • 事件 - 用户交互的响应
  • DOM - 文档对象模型
