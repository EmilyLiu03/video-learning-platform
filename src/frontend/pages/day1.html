🌟Day 1：HTML、CSS、JavaScript 基础 - 前端魔法的三大支柱

欢迎来到前端魔法的世界！今天我们将学习构建网页的三大核心技术：HTML、CSS 和 JavaScript。这三种技术就像魔法的三大支柱，HTML 构建结构，CSS 美化外观，JavaScript 赋予生命力。

💡今日核心魔法

  • HTML 结构魔法：学会用标签构建网页骨架
  • CSS 样式魔法：掌握美化网页的艺术
  • JavaScript 交互魔法：为网页注入动态生命力
  • 三者融合：创建第一个完整的网页作品

🧰准备工作

在开始今天的魔法修炼之前，请确保你有一个文本编辑器（如 VS Code、Sublime Text 或 Notepad++）和一个现代浏览器（Chrome、Firefox、Safari 或 Edge）。

🏗️第一节：HTML - 网页的骨架

HTML（HyperText Markup Language）是构建网页的基础语言。它使用标签来定义网页的结构和内容。

🔍基础HTML结构：

<!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>我的第一个网页</title> </head> <body> <h1>欢迎来到我的网页！</h1> <p>这是我用HTML创建的第一个段落。</p> </body> </html>

常用HTML标签

标签 作用 示例
<h1>-<h6> 标题标签 <h1>主标题</h1>
<p> 段落标签 <p>这是一个段落</p>
<div> 容器标签 <div>内容区域</div>
<a> 链接标签 <a href="https://example.com">链接</a>
<img> 图片标签 <img src="image.jpg" alt="描述">
<ul>/<li> 无序列表 <ul><li>项目1</li></ul>

🎨第二节：CSS - 网页的美妆师

CSS（Cascading Style Sheets）负责网页的视觉呈现。它可以控制颜色、字体、布局等所有视觉元素。

🔍CSS基础语法：

/* 选择器 { 属性: 值; } */ h1 { color: #2c3e50; font-size: 24px; text-align: center; } .container { width: 80%; margin: 0 auto; padding: 20px; background-color: #f8f9fa; } #header { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; padding: 20px; }

CSS选择器类型

选择器 语法 作用
标签选择器 h1 { } 选择所有h1标签
类选择器 .class { } 选择class="class"的元素
ID选择器 #id { } 选择id="id"的元素
后代选择器 div p { } 选择div内的所有p标签

✨CSS常用属性：

  • color：文字颜色
  • background-color：背景颜色
  • font-size：字体大小
  • margin：外边距
  • padding：内边距
  • border：边框
  • width/height：宽度/高度
  • display：显示方式

⚡第三节：JavaScript - 网页的魔法师

JavaScript 是一种编程语言，它为网页添加交互性和动态功能。从简单的按钮点击到复杂的动画效果，JavaScript 都能实现。

🔍JavaScript基础语法：

// 变量声明 let userName = "魔法师"; const magicPower = 100; // 函数定义 function castSpell(spellName) { return "施展了" + spellName + "魔法！"; } // 事件处理 document.getElementById('magic-btn').addEventListener('click', function() { alert(castSpell("火球术")); }); // DOM操作 document.querySelector('.title').textContent = "标题已被魔法改变！"; document.querySelector('.box').style.backgroundColor = "blue";

JavaScript核心概念

概念 说明 示例
变量 存储数据的容器 let name = "小明";
函数 可重复使用的代码块 function greet() { ... }
事件 用户交互的响应 click, hover, submit
DOM 文档对象模型 document.getElementById()

🎨第四节：HTML/CSS/JavaScript 三者融合

现在让我们将三种技术结合起来，创建一个真正的交互式网页！

✨魔法变色盒子

这是一个完整的示例，展示了HTML、CSS和JavaScript如何协同工作：

<!-- HTML结构 --> <div class="magic-box" id="colorBox"> <h3>点击我改变颜色！</h3> <button onclick="changeColor()">魔法按钮</button> </div> <!-- CSS样式 --> <style> .magic-box { width: 200px; height: 200px; background: linear-gradient(45deg, #ff6b6b, #4ecdc4); border-radius: 15px; text-align: center; padding: 20px; transition: all 0.3s ease; cursor: pointer; } .magic-box:hover { transform: scale(1.1); box-shadow: 0 10px 20px rgba(0,0,0,0.3); } </style> <!-- JavaScript交互 --> <script> function changeColor() { const colors = ['#ff6b6b', '#4ecdc4', '#45b7d1', '#96ceb4', '#feca57']; const randomColor = colors[Math.floor(Math.random() * colors.length)]; document.getElementById('colorBox').style.backgroundColor = randomColor; } </script>
